Projetos

  • Site AneTry

    Aplicação web Full-Stack para avaliação de jogos, ela conta com funcionalidades de CRUD completo, utilizando também o servidor XAMPP/Apache para o banco de dados MySQL gerenciado pelo PHPMyAdmin e o Bootstrap para personalização e responsividade.

  • Monitoramento de Estufa

    Projeto acadêmico, para monitoramento de temperatuda e umidade. Realizado com hardware físico ESP32Wroom, Relé 5v e o DHT11, via Thonny IDE no SO Ubuntu, juntamente com o ThingSpeak para atualização em tempo real da medição de temperatura.

  • Gerenciamento de Escola

    Sistema de Gerenciamento, contendo menus CRUD completos, permite controle de turmas e alunos.

  • Monitoramento Temperatura e Umidade

    Sistema de monitoramento IoT para estufas, integrando sensores de temperatura e umidade com dashboards mobile/web via protocolo MQTT e Blynk.

  • Mini-Game Dino

    Mini jogo que referencia o jogo do dino quando não se tem internet. Desenvolvido em HTML5, CSS3 e JavaScript.

  • Sistema Financiamento

    Sistema de Financiamento Imobiliário, focado em orientação a objetos, lógica de cálculo de juros e parcelas, desenvolvido em Java.
